# Herb Minifier

:::code-group

```shell [npm]
npm add @herb-tools/minifier
```

```shell [pnpm]
pnpm add @herb-tools/minifier
```

```shell [yarn]
yarn add @herb-tools/minifier
```

```shell [bun]
bun add @herb-tools/minifier
```
:::
